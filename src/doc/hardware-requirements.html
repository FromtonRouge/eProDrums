Hardware requirements

Electronic drum kit

To use eProDrums you need an electronic drum kit with a midi output. A standard midi out port looks like this (5-pin):

If you have this kind of output you'll need a midi-to-usb adapter to connect your drum module to your computer

On some drum modules, instead of a 5-pin port, you can find a usb port like this :

In that case, you don't need a midi-to-usb adapter to connect your drum module to the computer

Madcatz midi pro adapter or a Midi Rocker LX

To play Rock Band with your electronic drum kit, you need the Madcatz midi pro adapter.

It should also work with a Midi Rocker LX (http://www.bytearts.com/midi_lx)

Midi usb adapter

To connect your drum module to the computer you'll need a midi-to-usb adapter except if you have a usb output port (see above). If you plan to use a computer under linux, be sure to check the compatibility first, some midi-to-usb adapters work only on Windows.

Computer

eProDrums can be installed on a computer under the following OS

  • Windows : with the installer
  • Linux : by building eProDrums from sources

Note 1 : It should work on OS X if you build eProDrums from sources but I never tested it

Note 2 : eProDrums works well on the Raspberry Pi if you minimize the application (to disable GUI update). A Raspberry Pi cost 30$, so if you need a cheap computer to only run eProDrums it's a good choice.
