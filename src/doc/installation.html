Windows

  2. Install the software

Linux (debian/ubuntu/raspbian)

There is no package (don't know how to do that yet). So you'll have to build eProDrums from source.

  1. Get the source code from : www.github.com/FromtonRouge/eProDrums
  2. Get portmidi : sudo apt-get install libportmidi-dev
  3. Get Qt : sudo apt-get install qt4-qmake qt4-dev-tools qt4-qtconfig
  4. Get Qwt from source (at this time 27/01/2013, there is no Qwt6 dev package)
    • Download Qwt source code from : http://qwt.sourceforge.net
    • Edit qwtconfig.pri and change QWT_INSTALL_PREFIX like this : QWT_INSTALL_PREFIX = /usr
    • qmake
    • make
    • sudo make install
  5. Get Boost : sudo apt-get install libboost1.50-all-dev
  6. Build eProDrums
    • qmake
    • make
    • sudo make install

Note: on a Raspberry Pi (raspbian), the installation is really long (3 or 4 hours).
