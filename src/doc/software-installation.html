Software installation

Windows

  1. Get the latest Windows installer from : www.github.com/FromtonRouge/eProDrums
  2. Install the software

Linux

Debian distributions (debian/ubuntu/raspbian etc...)

There is no package (don't know how to do that yet). So you'll have to build eProDrums from source.

  1. Get the source code from : www.github.com/FromtonRouge/eProDrums
  2. Get portmidi : sudo apt-get install libportmidi-dev
  3. Get Qt : sudo apt-get install qt4-qmake qt4-dev-tools qt4-qtconfig
  4. Get Qwt from source (at this time 27/01/2013, there is no Qwt6 dev package)
    • Download Qwt source code from : http://qwt.sourceforge.net
    • Edit qwtconfig.pri and change QWT_INSTALL_PREFIX like this : QWT_INSTALL_PREFIX = /usr
    • qmake
    • make
    • sudo make install
  5. Get Boost : sudo apt-get install libboost1.50-all-dev
  6. Build eProDrums
    • qmake
    • make
    • sudo make install

Note: on a Raspberry Pi (raspbian), the installation and compilation is really long (3 or 4 hours).

Other distributions

  1. Get, build and install the portmidi library from source : http://sourceforge.net/apps/trac/portmedia/wiki/portmidi
  2. Get, build and install Qt from source : http://qt.digia.com
  3. Get, build and install Qwt from source : http://qwt.sourceforge.net
  4. Get, build and install Boost from source : http://www.boost.org
  5. Get, build and install eProDrums from source : http://www.github.com/FromtonRouge/eProDrums

Mac OS X

Not tested... eProDrums is portable (in theory). You should build it from source.
