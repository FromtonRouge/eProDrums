What is this software ?

This software is a bridge between your electronic drum kit and the Rock Band Madcatz Midi Pro Adapter.

It reads midi notes sent by your drum kit, transform those notes (if needed), and then sends new data to the Madcatz Midi Pro Adapter.

But why to transform midi notes ? The main purpose is to translate some real drum techniques (eg: flams, hi-hat close/half-open/open, ghost notes etc...) and produce valid Rock Band midi notes to avoid a choke...

A basic example are flams : in Rock Band, snare flams are charted in [red+yellow] pads (for a righty drummer), but if you want to do a real flam on your electronic snare you'll produce 2 red notes.

eProDrums can interpret those hits and transform them to a [red+yellow] notes. Obviously, it can do more things...

Who should use this software ?

You should use this software if :

  • You play with the sound of your electronic drum kit and thus you have to play as real as possible (flams, open hi-hat etc...)
  • You don't play with the drum kit sound but you want to keep your playing as real as possible (flams, open hi-hat etc...)
  • You just want to use the hi-hat feature (close/half/open)
  • You have a Roland VH-11 or VH-12 and want to use them with Rock Band (close/half/open)
  • You want to add ghost notes in the chart
  • You are just curious...

What do I need ?

You need :

  1. A computer under :
    • Windows
    • Linux
      • Debian
      • Ubuntu
      • Raspbian
  2. A midi to usb adapter. If you want to use the software on Linux be sure to buy a compatible cable
  3. The Madcatz Midi Pro Adapter
  4. Sticks...

As you can see above, eProDrums works on a Raspberry Pi (Raspbian). A Raspberry Pi cost 30$ so don't buy an expensive computer just for eProDrums.
